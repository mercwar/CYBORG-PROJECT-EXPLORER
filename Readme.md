## Getting Started

### 1. Clone the Repository
```
git clone https://github.com/mercwar/cyborg-explorer.git
cd cyborg-explorer
```

### 2. Configure Filters
Edit `config.ini`:

```
root=/your/project/path
extensions=.c,.h,.txt,.avis
min_size=0
avis_mode=1
```

### 3. Run the Explorer
Linux/Mac:
```
./explorer_run.sh
```

Windows:
```
explorer_run.bat
```
